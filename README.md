## English

A userscript that expands the capabilities for working with Gemini chat history by removing restrictions on editing old messages.

## Problem Description

In the original Gemini web interface, there is a native ability to change only your very last request. All previous messages remain locked forever.

## Solution

The Gemini Editor script completely unlocks the hidden chat functionality. It allows editing the contents of any request in the history of the current chat.

Main features:
* Text editing
* Adding / removing attachments
* Changing the model
* Selecting tools

## Installation

A userscript extension is required to use the script.

1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension (or Violentmonkey) for your browser.
2. Install the script itself using one of the methods below:
   * [Install via Greasy Fork](https://update.greasyfork.org/scripts/575169/Gemini%20Editor.user.js) (Recommended)
   * [Install via GitHub](https://github.com/ushan0v/gemini-editor/raw/main/gemini-editor.user.js)

## Usage

After installing the script, refresh the Gemini tab. When hovering the cursor over any previous message, an edit button will appear.

**Secret feature:** If you click the edit button while holding the `Ctrl` key, the native editing mode from the Gemini interface itself will be forcibly invoked.

## Русский

Пользовательский скрипт, который расширяет возможности работы с историей чата Gemini, снимая ограничения на редактирование старых сообщений.

## Описание проблемы

В оригинальном веб-интерфейсе Gemini нативно предусмотрена возможность изменять только свой самый последний запрос. Все предыдущие сообщения остаются заблокированными навсегда.

## Решение

Скрипт Gemini Editor полностью разблокирует скрытый функционал чата. Он позволяет редактировать содержимое любого запроса в истории текущего чата.

Основные возможности:
* Редактирование текста
* Добавление / удаление вложений
* Смена модели
* Выбор инструментов

## Установка

Для использования скрипта требуется расширение пользовательских скриптов.

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) (или Violentmonkey) для вашего браузера.
2. Установите сам скрипт одним из способов ниже:
   * [Установить через Greasy Fork](https://update.greasyfork.org/scripts/575169/Gemini%20Editor.user.js) (Рекомендуемый)
   * [Установить через GitHub](https://github.com/ushan0v/gemini-editor/raw/main/gemini-editor.user.js)

## Использование

После установки скрипта обновите вкладку Gemini. При наведении курсора на любое прошлое сообщение, будет появляться кнопка редактирования. 

**Секретная функция:** Если кликнуть на кнопку редактирования с зажатой клавишей `Ctrl`, будет принудительно вызван нативный режим редактирования от самого интерфейса Gemini.
